# Ikarus
Ikarus jest biblioteką Daedalusa - języka skryptowego Gothica. Wykorzystuje interpreter, aby umożliwić dowolny dostęp do pamięci i definiuje mnóstwo przydatnych funkcji do pracy z silnikiem.
